# **App Name**: DevSpace

## Core Features:

- Hero Section: Display a 3D glassmorphism icon and a brief introduction.
- Project Showcase: Present key projects such as 'AI Eye-Tracking System' and 'EduShare Portal'.
- Skills Display: List technical skills including Java, AWS, and React with logo representations.
- AI Layout Generation: Use an AI tool to generate initial UI layouts based on provided requirements.
- Contact Form: A form which can submit email to the developer

## Style Guidelines:

- Primary color: Bold Yellow (#FFDA63) to highlight key elements and interactive components.
- Background color: Deep Black (#121212) for contrast and a modern feel, emphasizing content visibility.
- Accent color: White (#FFFFFF) to create strong visual hierarchy and legibility against the dark background.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused appearance.
- Body font: 'Inter' (sans-serif) for clear and readable paragraphs of text.
- Implement minimalist, geometric icons suitable for a technical and professional aesthetic.
- Employ a neubrutalist design with high contrast, defined borders, and structured content sections to enhance readability and visual impact.
